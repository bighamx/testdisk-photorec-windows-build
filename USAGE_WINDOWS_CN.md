# Windows 使用说明

## 发布包类型

本仓库提供两种 Windows x64 发布包:

- Qt 图形版: 包含 `qphotorec.exe`
- 命令行版: 包含 `testdisk.exe`、`photorec.exe`、`fidentify.exe`

## 运行前准备

1. 从 GitHub Releases 下载发布包。
2. 将 zip 解压到普通目录，例如 `C:\Tools\TestDisk`。
3. 不要在压缩包内部直接双击运行程序。

## 主要程序

### `qphotorec.exe`

适合希望使用图形界面进行文件恢复的场景。

1. 启动 `qphotorec.exe`
2. 选择磁盘或镜像文件
3. 选择分区
4. 选择恢复文件的输出目录
5. 开始扫描和恢复

### `photorec.exe`

适合使用命令行界面的文件恢复场景。

1. 启动 `photorec.exe`
2. 选择磁盘或镜像文件
3. 选择分区和文件系统选项
4. 尽量把恢复目录选到另一块磁盘
5. 开始恢复

### `testdisk.exe`

适合分区分析、分区恢复和分区表修复场景。

1. 启动 `testdisk.exe`
2. 视需要创建日志
3. 选择目标磁盘
4. 选择分区表类型
5. 使用 `Analyse` 检查结果，确认无误后再执行写入

## 安全提示

- 恢复文件时尽量写入另一块磁盘或另一分区。
- 在恢复完成前，尽量不要向损坏磁盘写入新数据。
- 如果不确定，先做只读分析和日志收集。
- 分区修复会修改磁盘元数据，执行前请仔细确认。

## Qt 图形版包含内容

Qt 图形版发布包中包含:

- 可执行文件
- Qt 运行库
- MinGW 运行库
- 平台插件
- 图片格式插件

## 本构建的范围说明

这个 Windows 发布包的目标是“可直接运行和分发”。
它并不代表已经启用了上游项目的全部可选库。
