# Windows x64 Qt Build

This release provides a Windows x64 build of TestDisk / PhotoRec compiled with `MSYS2 + MinGW-w64 + Qt5`.

## Included binaries

- `testdisk.exe`
- `photorec.exe`
- `fidentify.exe`
- `qphotorec.exe`

## Notes

- Architecture: `x86_64`
- Qt runtime and platform plugins are bundled
- MinGW runtime dependencies are bundled
- This build focuses on a working Windows package and does not enable every optional filesystem library
